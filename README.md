# Editor de documentos

## Introdução
Essa é uma aplicação desenvolvida para edição de documentos em ambientes web colaborativos, permitindo que múltiplos usuários possam criar, editar e salvar documentos em tempo real. Dentre suas principais funcionalidades, estão:

1. Edição de texto em tempo real.
2. Exportação de documentos em diferentes formatos (PDF, DOCX, TXT).
3. Controle de versões, permitindo que os usuários revertam para versões anteriores do documento.
4. Suporte a comentários e anotações colaborativas.

## Tecnologias Utilizadas
- **Frontend**: React.js, ProseMirror, Redux
- **Backend**: Python, Django, Django Channels
- **Banco de dados**: SQLite (para desenvolvimento) e PostgreSQL (para produção)
